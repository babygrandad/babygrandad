## 📖 About Me

👋 Hey! I'm an aspiring full-stack developer with a backend bias 👀 most of the time. I like building clean, efficient APIs. I like building practical systems that bring solutions to real problems—but sometimes I’ll dabble just because I’m curious 🫠. The world has too many problems, and sometimes I just want to animate a div, you know?

I work primarily with .NET and Node/Express on the backend, though I’ve also spent time with PHP. SQL is my go-to for data persistence. I focus on writing code that’s lightweight, maintainable, and easy to reason about.

On the frontend, I lean on React—paired with Bootstrap, Tailwind, or even plain CSS 😎. I’ve worked with templating engines like EJS, but React just makes more sense for the kind of interfaces I like to build. I’m also looking to dip my toes into Next.js and explore what it can offer for full-stack development.
